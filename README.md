# Goblin Game - Ударь гоблина! 🔨

[![Build and Deploy](https://github.com/ksanr/ahj3-1/actions/workflows/deploy.yml/badge.svg)](https://github.com/YOUR_USERNAME/goblin-game-final/actions/workflows/deploy.yml)

Игра, где нужно успеть ударить гоблина молотком, прежде чем он исчезнет!

## 🎮 Правила игры

- Гоблин появляется в случайной ячейке на **1 секунду**
- Нужно успеть кликнуть по ячейке с гоблином
- За каждый успешный удар начисляется **+1 очко**
- За каждый пропуск начисляется **+1 пропуск**
- При **5 пропусках** игра заканчивается

## 🎯 Особенности

- 🖱️ Кастомный курсор-молоток
- ✨ Анимации появления гоблина и удара
- 📊 Счётчик очков и пропусков
- 🔄 Кнопка перезапуска игры

## 🛠️ Технологии

- Webpack
- ES6+ Classes
- Jest (тестирование)
- GitHub Actions (CI/CD)
